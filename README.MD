# How to start playing?
* Install Pygame
* Download all the .py, .jpg., .wav, .png files and put them in the same directory
* Run the .py file

# Keyboard Configuration
| Key   | Usage      |
| ---   | ---:       | 
| SPACE | Shoot      |
| ↑     | Jump       |
| ←     | Move left  |
| →     | Move right |
